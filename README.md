# 👓 OptyNova

Responsywna strona internetowa dla salonu optycznego wykonana jako projekt portfolio Front-End Developera.

Projekt został stworzony z wykorzystaniem HTML, SCSS (SASS), metodologii BEM oraz JavaScript.

---

## 🚀 Live Demo

🔗 https://arekfem.github.io/optynova/

## 🛠️ Technologie

Projekt został wykonany przy użyciu:

- HTML5
- SCSS (SASS)
- BEM Methodology
- JavaScript
- Responsive Web Design (RWD)

## ✨ Funkcjonalności

- responsywny layout,
- mobile navigation,
- sekcja usług,
- semantyczny HTML,
- przejrzysta struktura kodu.
